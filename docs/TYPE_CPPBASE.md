# Falcon type bases in C++


| Falcon type      | C++ type                         |
| ---------------- | -------------------------------- |
| ```int```        | ```BigInt```                     |
| ```float```      | ```double```                     |
| ```str```        | ```std::string```                |
| ```list```       | ```object**```                   |
| ```tuple```      | ```object**```                   |
| ```dict```       | ```std::unordered_map<object*, object*>``` |
| ```bool```       | ```bool```                       |
| ```bytes```       | ```char*```                       |
| ```set```       | ```std::set<object*>```         |
